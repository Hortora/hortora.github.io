Getting Started

Install the Hortora garden skills and capture your first entry.

Prerequisites

Hortora garden skills run inside Claude Code — Anthropic's CLI for Claude. You'll need:

  • Claude Code installed and authenticated
  • Python 3.8 or later
  • macOS or Linux

Install the skill manager

Download the claude-skill installer from the soredium repository:

# Download the skill manager
$ curl -fsSL https://raw.githubusercontent.com/Hortora/soredium/main/scripts/claude-skill -o claude-skill

Install forage and harvest

forage handles session-time operations (capturing and searching). harvest handles maintenance (merging submissions and deduplication).

$ python3 claude-skill install forage
$ python3 claude-skill install harvest
Tip: Claude Code auto-discovers skills placed in ~/.claude/skills/ — no registration step needed after install.

Capture your first entry

During your next Claude Code session, when something non-obvious surfaces — a bug that took an hour to find, a technique worth remembering — Claude will proactively offer to capture it. You can also trigger it directly at any time:

# In any Claude Code session
> /forage CAPTURE

Claude will ask for the entry details: what technology, what type (gotcha, technique, or undocumented behaviour), and a description. The entry queues as a submission until you run harvest.
